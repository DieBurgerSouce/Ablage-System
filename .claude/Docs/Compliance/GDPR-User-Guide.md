## Ihre Datenschutzrechte

Als Benutzer des Ablage-Systems haben Sie folgende Rechte gemäß DSGVO:

| Recht | Artikel | Beschreibung |
|-------|---------|--------------|
| **Auskunft** | Art. 15 | Erfahren, welche Daten über Sie gespeichert sind |
| **Berichtigung** | Art. 16 | Falsche Daten korrigieren lassen |
| **Löschung** | Art. 17 | Daten löschen lassen ("Recht auf Vergessenwerden") |
| **Einschränkung** | Art. 18 | Verarbeitung vorübergehend stoppen |
| **Datenübertragbarkeit** | Art. 20 | Daten in maschinenlesbarem Format erhalten |
| **Widerspruch** | Art. 21 | Verarbeitung widersprechen |

## Welche Daten werden verarbeitet?

### Benutzerdaten

| Datenkategorie | Zweck | Speicherdauer |
|----------------|-------|---------------|
| E-Mail-Adresse | Anmeldung, Kommunikation | Bis Kontolöschung |
| Name | Personalisierung | Bis Kontolöschung |
| Passwort (verschlüsselt) | Authentifizierung | Bis Kontolöschung |
| Letzte Anmeldung | Sicherheit | 90 Tage |

### Dokumentendaten

| Datenkategorie | Zweck | Speicherdauer |
|----------------|-------|---------------|
| Hochgeladene Dokumente | OCR-Verarbeitung | Bis Löschung durch Benutzer |
| Extrahierter Text | Suche, Anzeige | Bis Dokumentlöschung |
| OCR-Metadaten | Qualitätssicherung | Bis Dokumentlöschung |

### Protokolldaten

| Datenkategorie | Zweck | Speicherdauer |
|----------------|-------|---------------|
| IP-Adresse | Sicherheit, Missbrauchsschutz | 30 Tage |
| Aktionsprotokoll | Audit, Nachvollziehbarkeit | 365 Tage |
| Fehlermeldungen | Fehlerbehebung | 90 Tage |

## Daten einsehen (Art. 15)

### Über die Benutzeroberfläche

1. Klicken Sie auf Ihr **Profilbild** (oben rechts)
2. Wählen Sie **"Einstellungen"**
3. Klicken Sie auf **"Datenschutz"**
4. Wählen Sie **"Meine Daten anzeigen"**

Sie sehen eine Übersicht aller gespeicherten Daten:

```

│  Meine Daten                                                │
├─────────────────────────────────────────────────────────────┤
│  📧 E-Mail: benutzer@firma.de                               │
│  👤 Name: Max Mustermann                                    │
│  📅 Registriert: 15.03.2024                                 │
│  🔐 Letzte Anmeldung: 08.01.2025 10:30                      │
│                                                             │
│  📄 Dokumente: 127                                          │
│  📊 Speicherplatz: 2.4 GB                                   │
│                                                             │
│  [Vollständigen Bericht herunterladen]                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Vollständiger Datenbericht

Klicken Sie auf **"Vollständigen Bericht herunterladen"** für einen umfassenden Export:

- Alle Kontodaten
- Liste aller Dokumente
- Aktivitätsprotokoll (letzte 365 Tage)
- Freigegebene Dokumente
- Exportiert als ZIP mit JSON-Dateien

**Bearbeitungszeit:** Sofort bis 72 Stunden (bei großen Datenmengen)

## Daten korrigieren (Art. 16)

### Profildaten bearbeiten

1. **Einstellungen** → **Profil**
2. Ändern Sie die gewünschten Felder:
   - Name
   - E-Mail-Adresse (erfordert Bestätigung)
   - Passwort
3. Klicken Sie **"Speichern"**

### Dokumentmetadaten korrigieren

1. Öffnen Sie das Dokument
2. Klicken Sie auf **"Bearbeiten"** (Stift-Symbol)
3. Korrigieren Sie:
   - Dateiname
   - Dokumenttyp
   - Tags
   - Extrahierten Text (OCR-Korrektur)
4. Klicken Sie **"Speichern"**

### Korrekturanfrage stellen

Falls Sie Daten nicht selbst korrigieren können:

1. **Einstellungen** → **Datenschutz** → **Korrekturanfrage**
2. Beschreiben Sie die zu korrigierenden Daten
3. Fügen Sie Nachweise bei (optional)
4. Absenden

**Bearbeitungszeit:** Maximal 30 Tage

## Daten löschen (Art. 17)

### Einzelne Dokumente löschen

1. Wählen Sie das Dokument
2. Klicken Sie auf **"Löschen"** (Papierkorb-Symbol)
3. Bestätigen Sie die Löschung

**Hinweis:** Gelöschte Dokumente werden zunächst in den Papierkorb verschoben.

### Papierkorb endgültig leeren

1. Öffnen Sie den **Papierkorb**
2. Klicken Sie **"Papierkorb leeren"**
3. Bestätigen Sie: **"Endgültig löschen"**

⚠️ **Warnung:** Endgültig gelöschte Dokumente können NICHT wiederhergestellt werden!

### Benutzerkonto löschen

1. **Einstellungen** → **Datenschutz** → **Konto löschen**
2. Geben Sie Ihr Passwort ein
3. Wählen Sie, was gelöscht werden soll:
   - [ ] Nur mein Konto (Dokumente werden anonymisiert)
   - [x] Konto und alle meine Dokumente
4. Bestätigen Sie mit **"Konto unwiderruflich löschen"**

**Was wird gelöscht:**
- Alle persönlichen Daten
- Alle hochgeladenen Dokumente (wenn gewählt)
- Alle Aktivitätsprotokolle
- Alle Freigaben

**Was bleibt erhalten:**
- Anonymisierte Audit-Logs (gesetzliche Pflicht)
- Aggregierte Statistiken (ohne Personenbezug)

**Bearbeitungszeit:** Sofort (Deaktivierung), 30 Tage (endgültige Löschung)

## Daten exportieren (Art. 20)

### Datenexport anfordern

1. **Einstellungen** → **Datenschutz** → **Daten exportieren**
2. Wählen Sie das Format:
   - **JSON** (maschinenlesbar)
   - **CSV** (für Tabellenkalkulation)
   - **PDF** (zur Archivierung)
3. Wählen Sie, welche Daten exportiert werden sollen:
   - [x] Profildaten
   - [x] Dokumente
   - [x] Extrahierte Texte
   - [x] Aktivitätsprotokoll
4. Klicken Sie **"Export starten"**

### Export-Inhalt

```
export_2025-01-08.zip
├── profil.json                 # Kontodaten
├── dokumente/
│   ├── manifest.json           # Dokumentliste
│   ├── dokument_001.pdf        # Originaldateien
│   ├── dokument_001_text.txt   # Extrahierter Text
│   └── ...
├── aktivitaeten.json           # Protokoll
└── metadaten.json              # Export-Informationen
```

### Format: JSON (Beispiel)

```json
{
  "export_datum": "2025-01-08T10:30:00Z",
  "benutzer": {
    "id": "usr_abc123",
    "email": "benutzer@firma.de",
    "name": "Max Mustermann",
    "registriert": "2024-03-15T09:00:00Z"
  },
  "dokumente": [
    {
      "id": "doc_xyz789",
      "dateiname": "rechnung_2025-01.pdf",
      "hochgeladen": "2025-01-05T14:22:00Z",
      "dokumenttyp": "invoice",
      "extrahierter_text": "Rechnung Nr. 12345..."
    }
  ],
  "aktivitaeten": [
    {
      "zeitpunkt": "2025-01-08T10:15:00Z",
      "aktion": "document.upload",
      "details": "rechnung_2025-01.pdf"
    }
  ]
}
```

**Bearbeitungszeit:** Sofort bis 72 Stunden

## Verarbeitung einschränken (Art. 18)

Sie können die Verarbeitung Ihrer Daten vorübergehend einschränken, wenn:

- Sie die Richtigkeit der Daten bestreiten
- Sie die Daten für Rechtsansprüche benötigen
- Sie Widerspruch eingelegt haben (Prüfung läuft)

### Einschränkung beantragen

1. **Einstellungen** → **Datenschutz** → **Verarbeitung einschränken**
2. Wählen Sie den Grund
3. Geben Sie Details an
4. Absenden

**Während der Einschränkung:**
- Dokumente werden nicht für OCR verarbeitet
- Suche funktioniert weiterhin für bestehende Daten
- Keine automatischen Verarbeitungen

**Aufhebung:** Automatisch nach Klärung oder auf Ihren Wunsch

## Widerspruch einlegen (Art. 21)

### Widerspruch gegen Verarbeitung

1. **Einstellungen** → **Datenschutz** → **Widerspruch einlegen**
2. Wählen Sie die betroffene Verarbeitung:
   - OCR-Verarbeitung
   - Automatische Klassifizierung
   - Statistische Auswertung
3. Begründen Sie Ihren Widerspruch
4. Absenden

**Bearbeitungszeit:** 30 Tage

### Was passiert nach dem Widerspruch?

- Die betroffene Verarbeitung wird gestoppt
- Bestehende Daten bleiben erhalten
- Sie können den Widerspruch jederzeit zurückziehen

## Häufige Fragen

### Wie lange werden meine Daten gespeichert?

| Datenart | Speicherdauer |
|----------|---------------|
| Kontodaten | Bis Kontolöschung |
| Dokumente | Bis Sie sie löschen |
| Papierkorb | 30 Tage |
| Aktivitätsprotokolle | 365 Tage |
| Sicherheitslogs | 90 Tage |

### Werden meine Dokumente in der Cloud gespeichert?

**Nein.** Das Ablage-System ist eine On-Premises-Lösung. Alle Daten bleiben auf den Servern Ihrer Organisation.

### Wer hat Zugriff auf meine Dokumente?

- **Sie** haben vollen Zugriff auf Ihre Dokumente
- **Administratoren** haben keinen Zugriff auf Inhalte
- **Von Ihnen freigegebene** Personen
- **Niemand** sonst

### Kann ich sehen, wer auf meine Dokumente zugegriffen hat?

Ja, im Aktivitätsprotokoll:
1. Dokument öffnen
2. **"Aktivität"** Tab wählen
3. Alle Zugriffe werden mit Zeitstempel angezeigt

### Was passiert mit meinen Daten bei einer Datenpanne?

Bei einer Datenschutzverletzung:
1. Werden Sie innerhalb von 72 Stunden informiert
2. Erhalten Sie Details zur Verletzung
3. Bekommen Sie Handlungsempfehlungen
